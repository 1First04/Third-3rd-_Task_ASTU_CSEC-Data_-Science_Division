# 🧮 CLI Calculator (Python)

A simple yet powerful **Command-Line Interface (CLI) Calculator** built using Python. This project demonstrates core programming concepts such as functions, loops, conditional logic, and error handling in a clean and modular way.

---

## 📌 Features

* ✅ Basic arithmetic operations: `+`, `-`, `*`, `/`
* ✅ Power operation (`^`)
* ✅ Exponential calculation (`e^x`)
* ✅ Continuous execution until user exits
* ✅ Input validation and error handling
* ✅ Clean and modular function-based design

---

## 🧠 Concepts Used

* Functions and modular programming
* Conditional statements (`if`, `elif`, `else`)
* Infinite loops (`while True`)
* Exception handling (`try-except`)
* User input handling
* Python `math` module

## 🎮 Usage Instructions

1. Enter the first number
2. Choose an operator:

   * `+` → Addition
   * `-` → Subtraction
   * `*` → Multiplication
   * `/` → Division
   * `^` → Power
   * `exp` → Exponential (`e^x`)
3. Enter the second number (if required)
4. View the result
5. Type `exit` to quit the program
